# Sistema de Login em Java

Projeto simples desenvolvido em Java para treinar lógica de programação.

## Funcionalidades
- Solicita senha do usuário
- Permite até 3 tentativas
- Bloqueia após 3 erros
- Libera acesso se senha correta

## Tecnologias utilizadas
- Java
- IntelliJ IDEA
